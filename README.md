# 🎯 Number Guessing Game (Python)

A simple terminal-based number guessing game built using Python.

The player selects a difficulty level and tries to guess the randomly generated number within limited attempts.

---

## 🚀 Features

- 🎮 3 Difficulty Levels (Easy, Medium, Hard)
- 🎲 Random Number Generation
- 🏆 High Score System
- 🔁 Replay Option
- ⚠ Input Validation
- 📁 High score saved in file

---

## 🛠 Technologies Used

- Python
- Random Module
- File Handling
- OS Module

---

## 🎯 Difficulty Levels

| Level   | Range     | Attempts |
|----------|----------|----------|
| Easy     | 1–10     | 5        |
| Medium   | 1–20     | 3        |
| Hard     | 1–50     | 2        |

---

## ▶ How to Run

1. Install Python (3.x)
2. Download this repository
3. Open terminal in project folder
4. Run:

```
python game.py
```

---

## 🏆 High Score

The game automatically saves the best score in:

```
highscore.txt
```
